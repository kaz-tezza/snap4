画面上の任意の範囲のスクショを撮るソフトです。

【特徴】
　★ スクショ領域が次回も保持され、繰り返し同じ場所をスクショできます！
　★ マウスドラッグやカーソルキーで１ピクセル単位で範囲を微調整できます！
　★ スクショ領域の同一色部分の自動選択ができます！
　★ 白黒反転ができます！背景が黒いソフト画面のスクショに！

----------------------------------------------------------------------

［インストール方法］
　Zipファイルを任意のフォルダに展開するだけです。
　アプリと同じフォルダに設定(.ini)を保存するので書込権限のある場所に置いてください。

［アンインストール方法］
　インストールで展開したフォルダをエクスプローラ上で削除してください。


【使用方法】
　1. アプリケーション「snap4.exe」を起動しておきます。※１
　2. タスクバーにアイコンが追加されます。
　3. Ctrl+PrintScreenキーにてスナップショットモードになります。※２
　4. マウスドラッグで取得したい範囲を指定します。
　5. 範囲の右下に表示されるコピーアイコンをクリックするとクリップボードにコピーされます。
　6. 範囲の右下に表示されるセーブアイコンをクリックすると保存先指定ダイアログが表示されます。

　※１　オプションでスタートアップ時に自動起動する設定が可能です。
　※２　オプションで任意のキーに変更できます。

【機能説明−タスクバーメニュー】

タスクバーのアイコンをクリックすると、下記のメニューが表示されます。
・Snap Shot         : スナップショットを開始します。
・Self Timer        : 時間差でスナップショットを開始します。
・Option            : オプション設定ウインドウを表示します。
・Open SaveFolder   : 自動保存先フォルダーをエクスプローラで開きます(オプションで設定済みのこと)。
・Help              : readme.txtを表示します。
・About             : バージョン情報を表示します。
・Exit Application  : タスクトレイの常駐を終了します。


−−−−−−−−−−−−−−−−−−−−−−−−−−−−−−−−−−−−−−−−
【機能説明−スクショ領域のマウス操作】

スナップショットモード（画面が白っぽくなります）の状態で、スクショ開始できます。
＜操作：スクショ領域外＞
・マウスドラッグ           : 長方形のスクショ領域指定。
・Shift＋マウスドラッグ    : 正方形でのスクショ領域指定。
・ドラッグ中にカーソルキー : マウスポインタ位置を上下左右に微調整。
・カーソルキー             : スクショ領域の位置を微調整。領域が無い場合マウスポインタ位置。
・Shift＋カーソルキー      : スクショ領域の大きさを微調整。
・Ctrl+カーソルキー        : マウスポインタ位置を上下左右に微調整。
・中ボタン                 : 同色の領域を自動選択。
・Shift+中ボタン           : 同色の領域を追加で自動選択。

＜操作：スクショ領域内＞
・マウスドラッグ           : スクショ領域を自由移動。
・Shift＋マウスドラッグ    : スクショ領域を上下または左右固定で移動。

＜操作：スクショ領域の端のドラッガ（つまみ）＞
・マウスドラッグ           : スクショ領域のサイズ変更。
・Shift＋マウスドラッグ    : スクショ領域の縦横比率を維持しサイズ変更。
・Ctrl+マウスドラッグ      : スクショ領域の中央基準でサイズ変更。

【機能説明−ポップアップメニュー】
スナップショットモードのとき、
マウス右ボタンクリックすると下記のポップアップメニューが表示されます。

・Copy               : スクショ領域をクリップボードにコピーします。(Ctrl+C）
・Editor             : スクショ領域を外部エディタで開きます。(Ctrl+E)
・Save...            : スクショ領域をファイルに保存します。(Ctrl+S)
・Save and Continue  : スクショ領域を自動命名して保存しスクショを継続します。(Ctrl+Shift+S)　※３
 ---
・Close This Menu    : ポップアップメニューを閉じます
・Cancel             : スナップショットをキャンセルします。(Esc)

※３：オプション「AutoSave Folder」で設定したフォルダに「snap日付時間.png」の名前で保存します。


その他の機能
・Snapshot Areas
  ・Undo                  : スクショ領域の位置・大きさを1回前の状態に戻します(Ctrl+Z）
  ・Redo                  : Undoで戻した状態をやり直しします(Ctrl+Y)
  ・All Select            : デスクトップ全体をスクショ領域とします。（Ctrl+A）
  ・Clear Select          : スクショ領域をクリアします。(Ctrl+D)
  ・Pick Color Select     : クリック位置と同色の領域をスクショ領域に自動設定します(Space)
                            このモード中、Shiftキー＋左ボタンドラッグで範囲の拡張ができます。
  ・SpecifySize	          : スクショサイズを数値で入力します。(F2)
  ・Show Saved Areas      : 保存したスクショ領域位置を確認します。(0キーでOn/Off)
  ・#1 (x,y)-(x,y)        : スクショ領域位置を保存できます。（Ctrl+1〜5で保存、1〜5キーでロード)
                            外部ディスプレイの接続で位置が変わることがあります。
・Info Copy
  ・SnapArea              : スクショ領域の座標情報をクリップボードにコピーします。
  ・SnapSize              : スクショ領域のサイズ情報をクリップボードにコピーします。
  ・RRGGBB                : マウス位置の画素の色(#RRGGBB)をクリップボードにコピーします。
  ・R,G,B                 : マウス位置の画素の色(10進数)をクリップボードにコピーします。
・Tools
  ・Screen Update         : デスクトップ画面を再読み込みします。(F5)
  ・Show Magnifying Glass : マウスポインタ周辺の拡大鏡を表示します。(Zキー)
  ・Show SnapArea Size    : スクショ領域のサイズ値を左上に表示します。(Sキー)
  ・RealTime Screen Mode  : デスクトップをリアルタイム表示します。拡大鏡は利用できなくなります。(Ctrl+R)
  ・Popup Picture         : スクショ領域をポップアップwindowで表示します。一時的なメモ用途に。（Ctrl＋Shift+E）
・Filter
  ・Reverse B/W           : 黒と白を入れ替えてコピーします。(BキーでOn/Off)
  ・Nega                  : ネガ反転色でコピーします。(NキーでOn/Off)
  ・Border Line           : スクショ時に境界線を含みます。（ＯキーでOn/Off）
  ・Mouse Cursor          : マウスカーソルもスクショします。(Mキー)

【タスクトレイ　→　オプション設定ウインドウ】
タスクトレイメニューからOptionをクリックすると、オプション設定ウインドウが表示されます。
オプション設定中はスナップショットモードに移行できません。

・Snapshot Key
  スクショーモードを開始するショートカットキーを指定します。
  Ctrl、Shift、Alt、Win ＋ 任意のキーまたはPrintScreenキーの組み合わせが指定できます。
  すでに別のソフトやシステムのホットキーとして使われている場合は利用できません。
  

・Remember Previous Area
  ONのとき、スクショーモードが開始時点で前回のスクショ領域が表示されます。
  OFFのとき、スクショ領域はリセットされていますが、Undo (Ctrl+Z)で前回のスクショ領域を復元できます。

・Show Action (Copy, Save, Edit) Icons
  ONのとき、スクショ領域の右下に「コピー」「保存」などのアイコンボタンを表示します。

・Picture Editor
  スクショ領域の画像を外部エディタで開くための設定です。
  任意のエディタのアプリケーションへのパスを指定します。
  無効なパスの場合はエディタアイコンが表示されません。

・AutoSave Folder
  自動命名での保存先フォルダーを指定します。

・Reverse Black/White Threshold Color
  白黒反転の、黒または白とみなす色の敷居値を設定します。
  
・Execute Application on Startup
  Windows起動時に自動で起動するか設定します。

・OK、Cancelボタン
  設定を確定（OK）、または破棄（Cancel）してオプション設定ウインドウを閉じます。
