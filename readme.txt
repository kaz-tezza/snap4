【特徴】
　・デスクトップ画面の領域スクリーンキャプチャーを撮るソフトです。
　・キャプチャ領域が保持されるので、次回も繰り返し同じ場所をキャプチャできます。
　・キャプチャ領域を、マウスドラッグやカーソルキーで1ピクセル単位で微調整ができます。
　・同一色の領域を取得してキャプチャ領域を自動選択できます。
　・白黒反転や、ネガ反転のフィルタ機能があります。
　・クリップボードへコピーや、PNG/BMPファイルへ保存、画像ソフトでの編集が選択できます。


［インストール］
　Zipファイルを任意のフォルダに展開してください。
　ただし、アプリケーションと同じフォルダに設定(.ini)を書込むので書込み権限のある場所にしてください。
　Windowsサインイン時に自動で起動したい場合は、下記フォルダーにsnap4.exeのショートカットを作成してください。
　%APPDATA%\Microsoft\Windows\Start Menu\Programs\Startup\

［アンインストール］
　レジストリ登録などはしていません。
　インストールで展開したフォルダをエクスプローラ上で削除してください。


【使用方法】
　1. アプリケーション「snap4.exe」を起動しておきます。※１
　2. タスクバーにアイコンが追加されます。
　3. Ctrl+PrintScreenキーにてスナップショットモードになります。※２
　4. マウスドラッグで取得したい範囲を指定します。
　5. 範囲の右下に表示されるコピーアイコンをクリックするとクリップボードにコピーされます。
　6. 範囲の右下に表示されるセーブアイコンをクリックすると保存先指定ダイアログが表示されます。

　※１　スタートアップに登録して起動されると便利です。
　※２　オプションで任意のキーに変更できます。

【機能説明−タスクバーメニュー】

タスクバーのアイコンをクリックすると、下記のメニューが表示されます。
・Snap Shot         : スナップショットを開始します。
・Self Timer        : 時間差でスナップショットを開始します。
・Option            : オプション設定ウインドウを表示します。
・Open SaveFolder   : 自動命名での保存先フォルダーをエクスプローラで開きます(オプションで設定済みのこと)。
・Help              : readme.txtを表示します。
・About             : バージョン情報を表示します。
・Exit Application  : タスクバーの常駐を終了します。

【機能説明−キャプチャ領域のマウス操作】

スナップショットモード（画面が白っぽくなります）の状態で、キャプチャ開始できます。
＜操作：キャプチャ領域外＞
・マウスドラッグ           : 長方形のキャプチャ領域指定。
・Shift＋マウスドラッグ    : 正方形でのキャプチャ領域指定。
・ドラッグ中にカーソルキー : マウスポインタ位置を上下左右に微調整。
・カーソルキー             : キャプチャ領域の位置を微調整。領域が無い場合マウスポインタ位置。
・Shift＋カーソルキー      : キャプチャ領域の大きさを微調整。
・Ctrl+カーソルキー        : マウスポインタ位置を上下左右に微調整。
・中ボタン                 : 同色の領域を自動選択。
・Shift+中ボタン           : 同色の領域を追加で自動選択。

＜操作：キャプチャ領域内＞
・マウスドラッグ           : キャプチャ領域を自由移動。
・Shift＋マウスドラッグ    : キャプチャ領域を上下または左右固定で移動。

＜操作：キャプチャ領域の端のドラッガ（つまみ）＞
・マウスドラッグ           : キャプチャ領域のサイズ変更。
・Shift＋マウスドラッグ    : キャプチャ領域の縦横比率を維持しサイズ変更。
・Ctrl+マウスドラッグ      : キャプチャ領域の中央基準でサイズ変更。

【機能説明−ポップアップメニュー】
スナップショットモードのとき、
マウス右ボタンクリックすると下記のポップアップメニューが表示されます。

・Copy               : キャプチャ領域をクリップボードにコピーします。(Ctrl+C）
・Editor             : キャプチャ領域を外部エディタで開きます。(Ctrl+E)
・Save...            : キャプチャ領域をファイルに保存します。(Ctrl+S)
・Save and Continue  : キャプチャ領域を自動命名して保存しキャプチャを継続します。(Ctrl+Shift+S)　※３
 ---
・All Select         : デスクトップ全体をキャプチャ領域とします。（Ctrl+A）
・Clear Select       : キャプチャ領域をクリアします。(Ctrl+D)
・Pick Color Select  : クリック位置と同色の領域をキャプチャ領域に自動設定します(Space)
                            このモード中、Shiftキー＋左ボタンドラッグで範囲の拡張ができます。
・Close This Menu    : ポップアップメニューを閉じます
・Cancel             : スナップショットをキャンセルします。(Esc)

※３：オプション「AutoSave Folder」で設定したフォルダに「snap日付時間.png」の名前で保存します。


その他の機能
・Snapshot Areas
  ・Undo                  : キャプチャ領域の位置・大きさを1回前の状態に戻します(Ctrl+Z）
  ・Redo                  : Undoで戻した状態をやり直しします(Ctrl+Y)
  ・SpecifySize	          : キャプチャサイズを数値で入力します。(F2)
  ・Show Saved Areas      : 保存したキャプチャ領域位置を確認します。(0キーでOn/Off)
  ・#1 (x,y)-(x,y)        : キャプチャ領域位置を保存できます。（Ctrl+1〜5で保存、1〜5キーでロード)
                            外部ディスプレイの接続で位置が変わることがあります。
・Info Copy
  ・SnapArea              : キャプチャ領域の座標情報をクリップボードにコピーします。
  ・SnapSize              : キャプチャ領域のサイズ情報をクリップボードにコピーします。
  ・RRGGBB                : マウス位置の画素の色(#RRGGBB)をクリップボードにコピーします。
  ・R,G,B                 : マウス位置の画素の色(10進数)をクリップボードにコピーします。
・Others
  ・Screen Update         : 現在のデスクトップ状態に更新します。(F5)
  ・Show Magnifying Glass : マウスポインタ周辺の拡大鏡を表示します。(Zキー)
  ・Show SnapArea Size    : キャプチャ領域のサイズ値を左上に表示します。(Sキー)
  ・Show Mouse Cursor     : マウスカーソルもキャプチャします。(Mキー)
  ・RealTime Screen Mode  : デスクトップをリアルタイム表示します。拡大鏡は利用できなくなります。(Ctrl+R)
  ・Popup Picture         : キャプチャ領域をポップアップwindowで表示します。一時的なメモ用途に。（Ctrl＋Shift+E）
・Filter
  ・Reverse B/W           : 黒と白を入れ替えてコピーします。(BキーでOn/Off)
  ・Nega                  : ネガ反転色でコピーします。(NキーでOn/Off)
  ・Border Line           : キャプチャ時に境界線を含みます。（ＯキーでOn/Off）

【タスクトレイ　→　オプション設定ウインドウ】
タスクトレイメニューからOptionをクリックすると、オプション設定ウインドウが表示されます。
オプション設定中はスナップショットモードに移行できません。

・Snapshot Key
  キャプチャーモードを開始するショートカットキーを指定します。
  Ctrl、Shift、Alt、Win ＋ 任意のキーまたはPrintScreenキーの組み合わせが指定できます。
  すでに別のソフトやシステムのホットキーとして使われている場合は利用できません。
  

・Remember Previous Area
  ONのとき、キャプチャーモードが開始時点で前回のキャプチャ領域が表示されます。
  OFFのとき、キャプチャ領域はリセットされていますが、Undo (Ctrl+Z)で前回のキャプチャ領域を復元できます。

・Show Action (Copy, Save, Edit) Icons
  ONのとき、キャプチャ領域の右下に「コピー」「保存」などのアイコンボタンを表示します。

・Picture Editor
  キャプチャ領域の画像を外部エディタで開くための設定です。
  任意のエディタのアプリケーションへのパスを指定します。
  無効なパスの場合はエディタアイコンが表示されません。

・AutoSave Folder
  自動命名での保存先フォルダーを指定します。

・Reverse Black/White Threshold Color
  白黒反転の、黒または白とみなす色の敷居値を設定します。

・OK、Cancelボタン
  設定を確定（OK）、または破棄（Cancel）してオプション設定ウインドウを閉じます。
